# ðŸ§¾ FormGroup

A layout wrapper component for form fields, combining a label, child input, and optional helper text. Built for consistent structure across forms in the design system.

---

## âœ¨ Features

- Accepts any form input as children
- Optional `label` and `helperText` for context and accessibility
- Supports two label sizes: `default` and `large`
- Works with all input types (text, select, checkbox, radio, etc.)

---

## ðŸ“¦ Props

| Prop        | Type                     | Description                          |
|-------------|--------------------------|--------------------------------------|
| `label`     | `string`                 | The label displayed above the input  |
| `labelSize` | `'default' \| 'large'`   | Controls the size of the label text  |
| `helperText`| `string`                 | Optional helper text below the field |
| `children`  | `React.ReactNode`        | The input or field content           |
